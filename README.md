# Proyectx1

This is a 2D police simulator game. It has monochrome sprites, and it's rendered on GLS2, on Godot 3.3. 
Controls are:

W - Moving forward
S - Moving backwards
D - Moving right/Rotate car key
A - Moving left
E - Use car key
Space bar - Shoot/Slow down
Escape - Change scene
I - Inventory
